# Alekrypt V4

A advanced crypto analytics dashboard featuring a custom "Pump Engine", real-time DexScreener integration, and a Neo-Tokyo Cyberpunk UI.

## Features

*   **Pump Score Engine 3.0**: Institutional-grade scoring algorithm analyzing volume, liquidity, and smart money flow.
*   **Real-Time Data**: Live price integration via DexScreener API.
*   **Predictive Graph**: AI-driven trend projection based on Pump Scores.
*   **Risk Calculator**: Dynamic position sizing and stop-loss generation.
*   **Neo-Tokyo UI**: A fully immersive cyberpunk aesthetic with glitch effects and scanlines.

## Usage

Simply open `index.html` in any modern web browser. No build steps required.

## Live Data

The application fetches live data for the following tokens:
*   ETH, SOL, PEPE, WIF, BONK, BRETT, ARB, OP, AVAX, LINK.
